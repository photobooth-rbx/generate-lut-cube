# generate-lut-cube

This project is used to generate LUTs with roblox's grading system.

To use this you must open the project in studio with a custom 4096x4096 emulated device at actual resolution. Then you can save the project as a local plugin and run it. It will generate a LUT image where you can play around with lighting setting etc. When done, toggle the "Ready" attribute in workspace to get the output `.cube` file.
